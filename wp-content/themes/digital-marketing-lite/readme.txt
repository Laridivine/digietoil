For Homepage Setup:
====================

1. Go to Dashboard >> Pages >> Add New, Create a new page and name it as "Home" then select the template "Custom Home Page" and publish it.

2. Go to Dashboard >> Settings >> Reading, Select the option of Static Page, now select the page you created to be the homepage.

For Topbar Section:
======================

1. Go to Customizer >> Theme Settings >> Contact Details, add email address, phone number and publish it.

For Slider:
============

1. Go to Dashboard >> Pages >> Add New, Create a new Page and name it, add the content, featured image (image size 1400px x 660px) and publish the page. Repeat the step for other slider pages.

2. Go to Customizer >> Theme Settings >> Slider Settings, click the checkbox of Show/Hide Slider and select the pages of slider that you have created.

For About Section:
==================

1. Go to Dashboard >> Posts >> Add New, Create a new Post and name it, set it's featured image, add the content and publish the post.

3. Go to Customizer >> Theme Settings >> About Us, select the about post that you have created and publish it.

For Footer Section:
====================

1. Go to Dashboard >> Appearance >> Widgets, here you can add widgets to the footer widgets areas, which can be seen in footer.

For Copyright Text:
====================

1. Go to Customizer >> Theme Settings >> Footer Text, add footer text here.
